Entity Generator GUI

Windows form application that generates classes from database tables

How to run the code

* Add the stored procedure named "foreignKeyGet.sql" to your database.
* Run the code in visual studio
* once you run the code, pass the connection string parameter to the form input text, then click generate button to generate classes for your tables.

You can download the code from Github:

https://github.com/kiflay/EntityGeneratorGUI
